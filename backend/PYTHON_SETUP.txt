PYTHON EMAIL SETUP FOR DOST AMCen BOOKING FORM
==============================================

This system sends emails directly using Python (NO third-party services!)

REQUIREMENTS:
- Python 3.6+ (already installed on most computers)
- Gmail account: sv5323815@gmail.com

STEP-BY-STEP SETUP:

1. GENERATE GMAIL APP PASSWORD
   - Go to: https://myaccount.google.com/security
   - Click "App passwords" (if missing, enable 2-factor authentication first)
   - Select "Mail" and "Windows Computer"
   - Google generates a 16-character password
   - COPY THIS PASSWORD

2. UPDATE app.py
   - Open app.py in VS Code
   - Find line: GMAIL_PASSWORD = 'YOUR_GMAIL_APP_PASSWORD'
   - Replace 'YOUR_GMAIL_APP_PASSWORD' with your 16-character password
   - SAVE the file
   
   Example:
   GMAIL_PASSWORD = 'abcd efgh ijkl mnop'

3. INSTALL FLASK (one-time only)
   - Open PowerShell in your folder
   - Run: pip install flask flask-cors
   - Wait for installation to complete

4. START THE SERVER
   - In PowerShell, type: python app.py
   - You should see: "Server running at http://localhost:5000"
   - KEEP THIS WINDOW OPEN

5. TEST THE FORM
   - Open online booking.html in your browser
   - Fill out all required fields
   - Click "Submit Reservation"
   - Watch the PowerShell window - you should see "POST /submit 200"
   - Check your Gmail inbox - you should receive the booking email!

TROUBLESHOOTING:

Problem: "ModuleNotFoundError: No module named 'flask'"
Solution: Run: pip install flask flask-cors

Problem: "Error sending email: Login failed"
Solution: 
- Make sure you used the 16-character App Password, NOT your regular Gmail password
- Make sure 2-factor authentication is enabled on your Gmail account
- Make sure you copied the password correctly (no extra spaces)

Problem: "Address already in use"
Solution: 
- Another program is using port 5000
- Change the last line in app.py from "port=5000" to "port=5001"

Problem: Form shows "Send Failed"
Solution:
- Make sure the Python server is running (check PowerShell window)
- Make sure you opened the HTML file in a browser (not directly from file explorer)
- Check browser console (F12 → Console) for error messages

That's it! Your booking form now sends emails directly via Python! 🎉
